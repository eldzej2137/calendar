I used AI tools quite a lot for this project, especially as I had little experience with setting up a database. ChatGPT explained to me what I should do to connect a Spring Boot project with a PostgreSQL database and how to work with it, and with that knowledge I implemented the functionalities myself. Thus most of the code is written by me - including the SQL queries, all the Java code and the frontend page. However, I relied on AI's help to fully understand SQL, which was my main weakness. I asked ChatGPT some more questions whenever I was unsure about how something should work in the project.
To debug the project, if I could not find the problem myself, I asked GitHub Copilot. It helped be either by telling me what to change in my code or making the changes itself. One thing it actually did for me is set up Docker containerization, as it involves some fairly complicated "magic".
